# Makalu Everest Hospital Management System

## Overview
This repository contains the frontend implementation for the Reception and Admin panels of the Makalu Everest Hospital Management System. It was developed using **React** and **Tailwind CSS**.

## Features Implemented
- **Responsive and Interactive UI**: Optimized for cross-device compatibility.
- **Reception Panel**: Features for managing hospital reception tasks.
- **Admin Panel**: Dashboard for administrative oversight.
- **Patient Registration**: End-to-end forms and flows for registering new patients.
- **Appointment Management**: Interactive interfaces to schedule and manage appointments.
- **Medical Report Views**: Dynamic UI for viewing patient medical reports securely.
